# Сборка exe для распространения

## Требования

- Установленные **Node.js** (LTS) и **npm**
- Все зависимости установлены: в корне `desktop` и в папке `app`

## Как собрать exe

1. **Установить зависимости** (если ещё не установлены):

   ```bash
   cd c:\Projects\team-tracker\desktop
   npm install
   cd app
   npm install
   cd ..
   ```

2. **Собрать приложение** одной командой:

   ```bash
   npm run dist
   ```

   Команда сама соберёт React-приложение и затем создаст установщик и портативную версию.

3. **Результат** появится в папке **`dist`**:

   - **`Большой Брат Setup 1.0.0.exe`** — установщик (рекомендуется раздавать этот файл)
   - **`Большой Брат 1.0.0.exe`** — портативный exe без установки

## Важно для получателей

- Приложение подключается к **серверу по адресу http://localhost:5000**. Перед запуском exe сервер должен быть запущен на целевом компьютере или пользователи должны иметь доступ к серверу (тогда нужно будет изменить адрес API в коде и пересобрать).
- Для работы нужна **Windows 64-bit**.
